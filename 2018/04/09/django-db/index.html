django 数据库操作

Posted on 2018-04-09 | In python | | Visitors:

django1.7以后的版本，数据库迁移只有三个命令

  • migrate：用来迁移数据库
  • makemigrations：用来检测数据库变更和生成数据库迁移文件。
  • sqlmigrate：用来把数据库迁移文件转换成数据库语言(displays the SQL statements for a migratioin.）
